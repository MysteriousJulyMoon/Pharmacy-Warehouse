# Pharmacy-Warehouse
Pharmacy warehouse information system

Система может использоваться в аптеке или медицинском центре для автоматизации учета лекарств. Она заменяет Excel-таблицы и бумажные журналы, ускоряет поиск и исключает ошибки при работе с данными.

Интерфейс аптечного склада написан на Python с использованием Tkinter. 

В нем реализованы:

1. Просмотр всех лекарств в таблице (ID, название, цена, количество).

2. Добавление новых препаратов.

3. Редактирование цены и количества.

4. Удаление (мягкое — запись помечается как неактивная, но не удаляется физически).

5. Поиск по названию (регистронезависимый).

6. Фильтр «Только в наличии».

7. Просмотр противопоказаний для выбранного лекарства.

8. Статистика в реальном времени:

общее количество лекарств,

суммарное количество единиц на складе,

общая стоимость товаров.
